# 导出文件名称中文化

controller需要导出文件时, 指定中文文件名称
如果有表注释, 名称为`表注释 + 导入模板.xlsx`
如果没有表注释, 名称为`实体类名称 + 导入模板.xlsx`
导出同理
如果有表注释, 名称为`表注释 + 日期.xlsx`, 
如果没有表注释, 名称为`实体类名称 + 日期.xlsx`
其中日期格式, 我初步想法是 `yyyy-MM-dd HH:mm:ss`,
你建议什么格式
目的是让用户能区分导出的文件, 而不是因为名称重复变成xxx(1).xlsx这种
